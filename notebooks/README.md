# Notebooks

This folder contains the R Markdown source file used for the complete churn prediction analysis.

The analysis includes:

* Data cleaning and preparation
* Exploratory data analysis
* Feature evaluation
* Logistic Regression
* Naive Bayes
* Linear Discriminant Analysis (LDA)
* Quadratic Discriminant Analysis (QDA)
* Decision Tree modeling
* Model evaluation using ROC and AUC
* Customer churn scoring and retention analysis

Primary file:

* customer_churn_analysis.Rmd
