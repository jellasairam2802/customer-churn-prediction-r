# Data

This project uses a telecommunications customer churn dataset containing 6,488 customer records and 17 customer behavior, service usage, and demographic variables.

The dataset was used to identify customer churn drivers, evaluate classification models, and support retention strategy recommendations.

Key variables include:

* Customer tenure
* Contract type
* Monthly charges
* Internet service
* Online security
* Payment method
* Churn status

The original dataset is not included due to size and licensing considerations.
